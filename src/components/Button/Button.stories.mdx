## Colors

A button's color can be changed by passing a `color` prop. Any valid CSS color value is allowed.

The default color is set to the CSS variable --color-primary.

<Canvas>
  <Story name="Colors">

      <Button color="CadetBlue"/>

    </div>
  </Story>
</Canvas>

## Sizes

A button's size can be changed by passing a `size` prop. You can use any of the following values:
'xs', 'sm', 'md', 'lg', 'xl'

The default size is 'md'.

  <Story name="Sizes">

      <Button size="lg"/>
      <Button size="xl"/>
    </div>
  </Story>
</Canvas>

## Variants

A button's variant can be changed by passing a `variant` prop. You can use any of the following values:
'solid', 'outline', 'ghost', link'

The default variant is 'solid'.
